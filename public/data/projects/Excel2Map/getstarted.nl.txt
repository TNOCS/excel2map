# Beginnen met Excel2Map
### Je eigen  kaart maken
Download om te beginnen een Excel-bestand waarmee je een kaart kunt maken, bijvoorbeeld: <a href="data/excel2map_online_adressen.xlsm" target="_blank">voorbeeld Excel template</a>. Pas de data, namen en stijlen naar wens aan. Wanneer je daar klaar mee bent, kun je de data met de ' Upload to map '-knop vanuit het Excel-bestand naar een kaart exporteren. Om de kaart die je zojuist gemaakt hebt te bekijken, kun je klikken op de ' Show map '-knop in het Excel-blad.
### Handleiding
Een handleiding met de beschikbare parameters in de Excel template kan hier worden gedownload: <a href="data/excel2map.pdf" target="_blank">Excel2map handleiding</a>. Voor meer informatie over de functionaliteit van de interactieve kaart, neem een kijkje op de <a href="https://github.com/TNOCS/csWeb/wiki" target="_blank">csWeb wiki</a>. Hier kun je ook de contactgegevens van de ontwikkelaars vinden.
